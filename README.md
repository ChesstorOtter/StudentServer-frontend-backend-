# StuDoNline — Онлайн-школа для студентов

Веб-приложение для обучения студентов. Платформа предоставляет доступ к бесплатным и премиум-курсам после подтверждения студенческого статуса.

## Требования

- Docker версии 20.10 или выше
- Docker Compose версии 2.10 или выше

## Установка и запуск

1. Клонируйте репозиторий:
   git clone https://github.com/ваш-пользователь/studonline.git
   cd studonline

2. Запустите приложение с помощью Docker Compose:
   docker compose up --build


3. Откройте в браузере:
   http://localhost:3000


## Тестовые учётные записи

После запуска вы можете войти, используя следующие данные:

**Администратор**

## Структура проекта

- `frontend/` — HTML, CSS и JavaScript-файлы клиентской части.
- `backend/` — Исходный код сервера на языке Go.
- `init.sql` — SQL-скрипт для инициализации базы данных.
- `docker-compose.yml` — Конфигурация для запуска приложения в Docker.
- `Dockerfile` — Инструкции для сборки образа сервера.

## Остановка приложения

Для остановки приложения нажмите `Ctrl+C` в терминале, где запущен `docker compose`.

Чтобы полностью удалить данные и контейнеры, выполните:
docker compose down -v
